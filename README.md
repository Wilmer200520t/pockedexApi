## Installation

```bash
# Install node dependencies
npm install
```

## Run database

```bash
# Run mongo container
docker-compose up -d
```

## Running the app

```bash
# Start app as development
npm run start
```

```bash
# Start app as watch mode
npm run start:dev
```

```bash
# Start app as production mode
npm run build
npm run start:prod
```

- MongoDb
